# 🇮🇳 LexBharat — AI Legal Assistant for Indian Laws

LexBharat is an AI-powered legal assistant that answers questions based on Indian laws using RAG (Retrieval-Augmented Generation) with a vector database.  
Phase 1 focuses on: Indian Constitution, Indian Penal Code (IPC), and Motor Vehicle Act.

## Project Status
🔹 Stage: Setup & Dataset collection  
🔹 Current Focus: Raw data preparation
